# Contributing to Alchemy Screener

Thank you for considering contributing to the Alchemy Screener project! We welcome contributions from everyone.

## Getting Started
To get started with contributing:
- **Fork the repository**: Click on the fork button in the top right corner of this page.
- **Clone your fork**: Use `git clone <your-fork-url>` to clone your forked repository.
- **Set up the upstream remote**: This will help you keep your fork up to date.
  ```bash
  git remote add upstream https://github.com/Kasi72/alchemyscreener.git
  ```

## Development Workflow
1. **Create a new branch**: 
   ```bash
   git checkout -b <your-branch-name>
   ```
2. **Make your changes**: Edit the code, and ensure you follow the code style guidelines.
3. **Commit your changes**: Write a clear commit message explaining your changes.
   ```bash
   git commit -m "Describe your changes"
   ```
4. **Push your changes**: 
   ```bash
   git push origin <your-branch-name>
   ```
5. **Open a pull request**: Go to the original repository and click on "New Pull Request".

## Code Style
- Follow the existing conventions in the codebase.
- Use meaningful variable names and write clean, readable code.
- Ensure code is properly commented where necessary.

## Testing
- Ensure unit tests are written to cover new features or bug fixes. 
- Run existing tests to confirm that current functionalities remain intact:
  ```bash
  pytest # If you are using pytest, for example.
  ```

## Reporting Issues
If you encounter any bugs or issues:
- Check if the issue already exists by searching the [issues](https://github.com/Kasi72/alchemyscreener/issues) section.
- If not, provide a clear description of the bug, steps to reproduce, and any relevant screenshots. 

Thank you for helping us improve Alchemy Screener!
